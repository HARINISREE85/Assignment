Cascading style sheet

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language, such as HTML or XML. It is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

CSS enables the separation of content (structured in HTML) from its visual presentation.It provides equality for all browsers (Chrome, Edge, Firefox, Safari, Opera) on all devices (Desktop, Laptop, Tablet, Mobile).

CSS is free, open source, and uses standard CSS only (no JavaScript).
